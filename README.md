# Dynamic-Random
A website with dynamically changing stuff!   
By default Flikr will be used.  
For unsplash prepend: ?unsplash  
For a custom subject: ?subject=cats,mammal  

Code also used in https://github.com/NDevTK/Dynamic-Pigeons
